OSC経由でメッセージを送信するツール

## 機能
+ 内蔵ソフトウェアキーボードによりVRから簡単に日本語チャットが可能
+ 物理キーボードからソフトのテキストボックスに入力することで漢字変換も使用可

## 注意事項
一般VRChatterがAntigravityを使用して適当に作った実験的なツールです。
