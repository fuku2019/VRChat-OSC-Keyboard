## 機能・特徴
+ 内蔵のソフトウェアキーボードで、VRからひらがな・カタカナ・アルファベット入力が可能
+ 物理キーボードでソフトウェアのテキストボックスに入力すると、IME変換が使える
+ 美しいUIアニメーション
+ カスタムUIテーマ
+ モダンなデザイン

## ロードマップ
+ 漢字変換機能
+ カスタムレイアウト

## 注意事項
一般VRChatterがAntigravityを使って適当に作った実験的なツールです。
<br>
このソフトの使用によって発生したトラブルについては、自己責任でお願いします。
